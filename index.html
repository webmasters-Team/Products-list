This is a fake e-commerce website.

We made it as an use-case to demonstrate that Augmented Reality on the web is on its way to become functional and useful, thanks to the WebXR Device API.

Today, Android Chrome v81 supports WebXR by default. Hopefully other browsers will follow soon so that AR can be used everywhere without any plug-ins. For more information about WebXR, see this official explainer.
